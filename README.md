# LoRA‑адаптеры для генерации изображений волнистых попугайчиков 

## 1. Цель проекта

Разработать и обучить LoRA‑адаптеры для модели Stable Diffusion v1.5 с целью высококачественной персонализированной генерации изображений волнистых попугайчиков (budgerigars) в различных художественных стилях. Проект включает:

- Сбор и подготовку датасета из реальных фотографий.
- Обучение двух версий адаптера (на 27 и 63 изображениях) с последующим сравнением.
- Создание инференс‑пайплайнов с поддержкой Hydra‑конфигов, автоматической проверкой обрезания и множеством стилей.
- Документирование экспериментов и метрик.

---

## 2. Данные

| Версия | Количество изображений | Источник | Особенности |
|--------|------------------------|----------|-------------|
| v1     | 27                     | Собственная съёмка / отбор | Базовый набор для первичного обучения |
| v2     | 63                     | v1 + 36 новых кадров | Расширенный набор, включающий больше ракурсов, окрасок и деталей анатомии |

Все изображения приведены к разрешению 512×512, размечены единым промптом:  
`"a photo of a sks bird"` (токен `sks` используется для персонализации).

## 3. Модели и обучение

### 3.1 Архитектура

- Базовая модель: [`runwayml/stable-diffusion-v1-5`](https://huggingface.co/runwayml/stable-diffusion-v1-5)
- Тип адаптера: LoRA (Low‑Rank Adaptation) с рангом 16 и α = 16.
- Фреймворки: `diffusers`, `peft`, `accelerate`.

### 3.2 Гиперпараметры обучения

| Параметр               | Значение |
|------------------------|----------|
| Оптимизатор            | AdamW (8‑bit) |
| Скорость обучения      | 1e-4 |
| Batch size (эффективный)| 4 (batch=1, grad_acc=4) |
| Количество шагов       | 1500 |
| Микст точность         | fp16 |
| Регуляризация          | Dreambooth‑стиль с использованием instance‑изображений |

### 3.3 Сравнение версий

| Характеристика         | v1 (27 img) | v2 (63 img) |
|------------------------|-------------|-------------|
| Время обучения         | ~3 ч (CPU)  | ~15 мин (GPU T4) |
| Потеря (loss)          | 0.01–0.03   | 0.003–0.01 |
| Качество анатомии      | Обобщённая птица | Высокая детализация (восковица, волнистый рисунок, форма клюва) |
| Следование виду        | Среднее     | Высокое (≈90% генераций узнаваемы как волнистый попугай) |

## 4. Инференс и стилизация

### 4.1 Скрипты

- `inference.py` – универсальный пайплайн (может работать с любой LoRA).
- `inference_budgie_improved.py` – специализированный пайплайн с оптимизированными промптами для волнистых попугайчиков.

Оба скрипта поддерживают:
- Автоматическую проверку обрезания с повторной генерацией (до 3 попыток).
- Гибкое управление через Hydra (путь к LoRA, количество шагов, устройство и т.д.).
- Генерацию коллажа и сохранение метаданных.

### 4.2 Набор стилей

Реализовано 40+ стилей, сгруппированных по художественным направлениям:

- Пастельные и нежные (лаванда, сахарная вата, лунная нежность)
- Яркие и энергичные (радуга, солнечный свет, фестиваль)
- Динамичные (полёт, вихрь, скорость)
- Драматичные и тёмные (гроза, тень, ночной охотник)
- Романтические и медитативные
- Магические и сюрреалистические
- Специальные портретные с акцентом на детали вида (восковица, волнистый рисунок, пятна на щеках)

Каждый стиль сопровождается позитивным и негативным промптом, адаптированным под особенности волнистых попугайчиков.

## 5. Инструкция по запуску

### 5.2 Генерация изображений (v2)

bash
python inference_budgie_improved.py \
    trainer.device=cuda \
    inferencer.ckpt_dir="saved/lora_budgie" \
    inferencer.num_inference_steps=45
Параметры:

trainer.device – cuda или cpu

inferencer.ckpt_dir – путь к папке с LoRA (должна содержать adapter_model.safetensors)

inferencer.num_inference_steps – количество шагов диффузии (рекомендуется 30–50)

5.3 Использование старой модели (v1)
bash
python inference.py \
    trainer.device=cuda \
    inferencer.ckpt_dir="saved/lora_old"
5.4 Обучение новой LoRA (опционально)
bash
python train.py \
    trainer.device=cuda \
    datasets.train.dreambooth.data_path="/path/to/dataset" \
    trainer.save_dir="saved/lora_new" \
    trainer.num_epochs=5 \
    trainer.learning_rate=1e-4
6. Результаты экспериментов
6.1 Количественная оценка
Модель	Текстуальное сходство ↑	Визуальное сходство (DINOv2) ↑
Базовый SD 1.5	0.52	0.31
LoRA v1 (27)	0.67	0.58
LoRA v2 (63)	0.81	0.79
Текстуальное сходство – косинусное сходство эмбеддингов CLIP между промптом и изображением.
Визуальное сходство – среднее косинусное расстояние эмбеддингов DINOv2 между сгенерированным и реальным изображением.

6.2 Качественный анализ
Увеличение датасета с 27 до 63 образцов позволило модели выучить тонкие анатомические детали: цвет восковицы (синий/розовый), чёрно-жёлтый волнистый рисунок на голове, характерные пятна на щеках, форму клюва и длину хвоста.

Обучение на GPU ускорило итерации, что позволило быстро подобрать оптимальную скорость обучения и количество шагов.

Добавление в инференс‑промпты специфических ключевых слов (blue cere, scalloped markings, cheek patches) дополнительно усилило соответствие виду.
